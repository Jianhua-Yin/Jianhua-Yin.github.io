Identifying Quality EXAFS Data Fitting: Statistical Methods

Posted on 10-03-2022 Edited on 05-03-2023 In Core level spectrum , XAFS

Several criteria for determining the optimal fit are listed here, including statistical criteria, closeness of fit, and physical reasonableness.

Statistical Criterion

Number of Independent Points

Resolve peaks in the FT \(\Delta R=\frac{\pi}{2\left(k_{\max }-k_{\min }\right)}\)
\(N_{\text {ind }}=\frac{\left(R_{\max }-R_{\min }\right)}{\Delta R}=\frac{2\left(k_{\max }-k_{\min }\right)\left(R_{\max }-R_{\min }\right)}{\pi}\) Stern Formula +2

  • ideally packed
  • a first-shell fit

Measurement Uncertainty

Reduced \(\chi^2\)

\(\chi_v^2=\frac{N_{i n d}}{v N} \sum_{i=1}^N \frac{\left(\text { data }_i-f i t_i\right)^2}{\varepsilon_{i \text { random }}^2}\)

When fitting the FT of EXAFS, this equation is applied to the real and imaginary parts of the FT, and the results added together to give \(\chi^2_v\). comparing two fits to the same data

  • different number or identity of parameters
  • different rang of FT

\(\chi^2_v\) should not be compared for two fits for which the measurement uncertainty is expected to be significantly different.

  • should not be used to compare two otherwise identically constructed fits that are made on different ranges of \(\chi(k)\) taken from the same data set
  • have no way to know that systematic error varies from one part of \(\chi(k)\) to another of from one sample to another

Closeness Criterion

R-Factor

\(R=\frac{\sum_{i=1}^N\left(\text { data }_i-f i t_i\right)^2}{\sum_{i=1}^N\left(\text { data }_i\right)^2}\)

  • as long as the signal-to-noise ratio (S/N) of the data is good, the R-factor of adequate fits can be expected to be not more than a few percent
R-Factor Interpretation
\(\textcolor{green}{<0.02}\) Good enough
\(0.02-0.05\) Model has some details wrong / Data with low quality. Nevertheless, consistent with a broadly correct model.
\(0.05-0.10\) Serious flaws in model / Data with very low quality
\(\textcolor{red}{>0.10}\) Model may be entirely incorrect

Hamilton Test

Property - only appropriate for comparing two fits on the same \(k\)- and \(R\)-ranges - There is no different between applying the Hamilton Test to \(\chi^2\) \(\chi^2_v\) \(R\)-factor - Changes in the theoretical standards used by the model can often be accounted for as if they were changes in parameters

Procedure - Compute the ratio of the \(R\)-factor, \(r=\frac{R_1-factor}{R_2-factor}\) - Take half of the lower bound of the number of degrees of freedom of the closer fit, \(a=\frac{N_{idp}-Free\, parameters}{2}\) - Take half of the number of free parameters that have to be added to switch from one fit to the other, \(b=\frac{Added free patameters}{2}\) - Calculate the regularized incomplete beta function, \(I_r(a, b)=\frac{\int_0^r t^{a-1}(1-t)^{b-1} d t}{\int_0^1 t^{a-1}(1-t)^{b-1} d t}\)
