Advanced Topics in EXAFS Analysis

Posted on 10-03-2022 Edited on 05-03-2023 In Core level spectrum , XAFS Word count in article: 1.7k Reading time ≈ 6 mins.

There are several advanced techniques and strategies in EXAFS analysis, including multi-weight fitting, multi-data fitting, and the application of constraint restraints, among others.

Overview

  • Multiple k-weight Simultaneous Refinement of the data using multiple values of k-weighting in the Fourier Transform
  • Multiple Data Sets Simultaneous Refinement of multiple data sets
  • Multiple FEFF Using multiple runs of the FEFF program to generate the theoretical data used in fitting model
  • Set Constraints At the core of an EXAFS fitting model lie the relationships imposed among the fitting parameters
  • Set Restraints Applying Imperfect Knowledge to Influence the Evaluation of Model Fits. (Interactive Fitting)
