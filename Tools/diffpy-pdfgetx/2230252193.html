diffpy.pdfgetx: Advanced Programs for Converting Powder Diffraction Data into Total Scattering Pair Distribution Functions

Posted on 2023-07-15 In Tools , diffpy.pdfgetx Waline: Word count in article: 656 Reading time ≈ 2 mins.

diffpy.pdfgetx3 is an advanced and powderful software for converting Powder Diffraction data to Total Scattering atomic Pair Distribution Funtions. In addtion, it can also help us to extrat the PDFs from small-angle scattering(SAS). The program include three command line programs PDFgetx3, PDFgetN3 and PDFgetS3 for processing X-ray, Nertron, and Small-angle diffraction data respectively.

Overview

PDF data analysis typically consists of three parts:

  • Obtaining one-dimensional diffraction data (e.g., using Fit2D, Dioptas)
  • Extracting the total scattering pair distribution function (PDF) from the diffraction data (e.g., using diffpy.pdfgetx)
  • Modeling analysis of the PDFs (e.g., using RMCProfile, DISCUS).
    diffpy.pdfgetx can readily process hundreds of powder diffraction patterns within a few seconds and
