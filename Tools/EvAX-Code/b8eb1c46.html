Evolutionary Algorithm for EXAFS analysis

Posted on 02-09-2024 Edited on 05-01-2024 In Tools , EvAX Code

EvAX code is designed for the RMC-like analysis of well-defined structures, such as nanocrystalline materials.
The EvAX code is a simulation-based tool designed for the analysis of Extended X-ray Absorption Fine Structure (EXAFS) data, specifically tailored for crystalline and nanocrystalline materials. This tool facilitates detailed studies of the atomic-scale structure within these materials, leveraging advanced simulation techniques to enhance the understanding of their properties.

Overview

EvAX is a software program designed for the analysis of Extended X-ray Absorption Fine Structure data, utilizing Reverse Monte Carlo and Evolutionary Algorithm methods. It is particularly effective for research on materials with well-defined structures, such as nanocrystalline materials.
Key features:

  • Analysis of multiple scattering effects (Multiple-scattering effects contain useful structural information.)
  • Evolutionary algorithm-based structure optimization (RMC and EV)
  • Usage of wavelet transform for EXAFS signal comparison (Morlet wavelet or Gabor transform)
  • Analysis of EXAFS data from several absorption edges simultaneously (EvAX code allows one to construct an unambiguous single structure model, consistent with EXAFS data, acquired at several absorption edges.)

The Significance of EvAX

Pair Distribution Distance Disorder

EvAX restricts atomic displacements to only small shifts from their initial positions. This constraint allows for the modeling of complex bond-length distributions and facilitates the analysis of changes in interatomic distances and structural disorder. However, it does not permit the refinement of Coordination Numbers. Coordination Numbers are determined by the initial structural model provided by the user and remain unchanged during the EvAX simulations. Consequently, EvAX is particularly effective in scenarios where coordination numbers are predetermined, such as in the analysis of disorder in crystalline materials or well-defined nanocrystalline materials.

  • The approximate structure of the material are known. (From XRD ...)
    • EXAFS to study the disorder effects and atomic thermal motion
  • There are several plausible structure models for (From DFT,Molecular Dynamics ...)
    • to find which of them fits better with the experimental EXAFS data

How EvAX works

In RMC simulation, the process begins with an arbitrary initial configuration of atoms within a cell of a specified size and shape, which may or may not incorporate periodic boundary conditions. The total EXAFS spectrum, denoted as , is calculated for this configuration. Subsequently, the atomic configuration is altered by randomly modifying the coordinates of one or several atoms, leading to a new configuration for which the total EXAFS spectrum, , is computed. The newly calculated EXAFS spectra, and , are then compared to the experimental spectrum . Based on this comparison, the new atomic configuration is either accepted or discarded. This procedure is repeated multiple times until the positions of the atoms in the cell minimize the difference between the theoretical and experimental EXAFS spectra.
