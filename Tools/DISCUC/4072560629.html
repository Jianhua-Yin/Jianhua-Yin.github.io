Modeling Pair Distribution Functions Using DISCUC

Posted on 08-07-2023 In Tools , DISCUC

DISCUS is a comprehensive program designed to generate disordered atomic structures and compute corresponding experimental data, such as single crystal diffuse scattering or the atomic pair distribution function (PDF). Key features include symmetry operations, Monte Carlo simulation capabilities, and the ability to generate domain structures, among others. More recent developments in DISCUS have expanded its functionality to include the simulation of nanoparticles.
