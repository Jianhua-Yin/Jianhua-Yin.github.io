Use Dioptas for Efficient 2D Powder Diffraction Data Reduction

Posted on 06-30-2023 In Tools , Dioptas

Dioptas is an awesome GUI program based on Python, designed for the fast data reduction of Powder X-ray Diffraction images. It provides the capability of calibrating, integrating, creating masks, showing multiple pattern overlays and display phases line positions.

Procedure

Overview

Dioptas has three different modules which can all be accessed by the lab indicators on the left side of the user interface: Calibration, Mask, Integration.

  • Calibration Module This module is used to calibrate the detector geometry.
  • Mask Module This module allows users to select regions that need to be excluded from the image integration.
  • Integration Module This module displays the image and integrated pattern. It can overlay different patterns and show the line positions of phases.

Load the image data of the calibrant to perform the calibration and refinement procedures. Usually, the calibrant is a compound with a known and stable crystal structure, such as LaB6, CeO2, Cr2O3, etc.

Calibration Procedure

The calibration procedure aims to correct the detector geometry and find the beam center.

Preparation

First, click the button located on the left side of the window to enter calibration mode. The wavelength and pixel width/height have to be defined based on the experimental setup and detector used.

Start values
Required Parameter Setting:

  • Calibrant (standard complex)
  • wavelength
  • pixel size (width, height)
  • Approximate Distance (the program will automatically refine the value)

Peak Picking

Peak Selection
After selecting the 'Calibration Parameter', Dioptas requires an initial guess for the ring positions to determine the correct geometry. Select several points on each diffraction ring to proceed. OK, now, left-click at the location where we believe the first diffraction ring appears. Calibration dose not need to start from the first diffraction ring.

Calibration does not need to start from the first diffraction ring, which enables calibration of detector geometries with the beam center being far outside of the area of the detector. But we ought to make sure that current peak number belongs to the corresponding clicked ring.

If the default automatic peak search is enabled, simply left-click the first diffraction ring as shown in this figure. Zooming in on the image will help accurately target the first ring.
If the automatic peak searching fails, we can try the following alternatives:

  • Perform the automatic search on the second, third, or subsequent rings.
  • Choose the single peak search option.
  • automatic peak search
  • single peak search

Left click the localtion of the fist diffraction rings. The peak search routine will automatically detect the remaining diffraction rings when we click the button, located at the bottom left.

Left-click on the location of a diffraction ring; the program will mark the maximum peak around the clicked area. The Search size determines the search area. We should select as many diffraction rings as possible. - one peak for one diffraction ring (select the automatic increase)

Calibrate and Refine

Calibrate

Calibrate
After the ring have been selected we can start the calibration procedure. button, located at the bottom left. When the procedure is finished, Dioptas will switch to the Cake Tab. We can click on the pattern table to check if all peaks in the pattern coincide with the phase line positions. Check whether the calibration is successful. - the cake lines are straight - all peak in the pattern are coincided with phase line position

Refinement Options

After calibration and refinement, Dioptas will automatically create a 360 degree cake image and an integrated pattern. Some options can be defined. For high-quality diffraction data, it is recommended to set the intensity minimum factor to about 1-1.5.
For other parameters, refer to the manual.

Mask Creation

Create a mask to define areas that should be excluded from integration or calibration.

A mask excludes specific areas from the integration process, which enables the removal of unwanted features such as, for example, diamond reflections in diamond anvil cell experiments, saturated peaks or detector blemishes. Available geometries for creating masks are circles, rectangles and polygons.

  • mask
  • unmask

Mask

Choose a shape and left-click to set a start point, then drag the mouse to adjust the size of the shape, and finally left-click again to set the endpoint.

Integration Procedure

The core component of Dioptas where we can automatically integrate multiple images to pattern, browse between images and integrated pattern, compare multiple pattern to each other, perform background subtraction and compare pattern peak positions and intensities to the ones of known phases.

Exporting Data

Click Save to save the pattern as the desired file type. If we click the 'Save' button on the left, we will save the Dioptas project.
